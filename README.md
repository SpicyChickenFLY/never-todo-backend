# NEVER-TODO-BACKEND

[ Document in English ](./README.md)

## 总览

这是一个基于Gin框架的待办列表项目的后端部分，对应的前端项目由blue pongo负责进行开发（项目链接地址请点[这里](https://github.com/bluepongo/never-todo-frontend)）
该项目的目标是实现一个在线使用的支持自定义标签的个人代办列表功能，将来可能会实现离线使用的功能

## 特性：
* 待办列表功能
  * 支持新增、删除、修改代办
  * 支持为代办设置自定义标签
  * 支持按标签类型查找待办事项
